# 14-安全体系

## 📚 模块概述

本模块涵盖Java后端开发中的安全体系，包括认证授权、数据加密、安全防护等内容。安全是系统设计的重要组成部分。

## 🎯 学习目标

- 掌握OAuth2.0和JWT认证
- 理解HTTPS和SSL/TLS
- 掌握常见安全漏洞的防护
- 能够设计安全的系统架构
- 掌握数据加密和脱敏

# 11-补充知识

## 📚 模块概述

本模块包含Java后端开发的重要补充知识，这些知识是成为优秀Java开发者的必备技能。虽然不属于特定框架或技术栈，但在实际开发中至关重要。

## 🎯 学习目标

- 掌握Java并发编程的核心概念和实践
- 理解并应用23种设计模式
- 熟练使用常用数据结构和算法
- 掌握网络编程基础和Socket编程
- 能够进行系统性能分析和优化

   - 递归和动态规划

### 阶段三：实战应用（2-3周）
1. 网络编程
   - TCP/UDP编程
   - HTTP编程
   - NIO编程

2. 性能优化
   - JVM调优
   - 代码优化
   - 数据库优化

## 💡 学习建议

1. **循序渐进**: 先掌握基础，再学习进阶内容
2. **理论结合实践**: 每学一个知识点都要动手实现
3. **刷题巩固**: 通过LeetCode等平台巩固算法知识
4. **阅读源码**: 学习Spring等框架中的设计模式应用
5. **实战应用**: 在实际项目中应用所学知识
6. **定期复习**: 建立知识体系，定期回顾

## ⚠️ 常见误区

1. **过度追求算法**: 算法重要但不是全部，要平衡学习
2. **死记硬背**: 理解原理比记住代码更重要
3. **忽视基础**: 并发、网络等基础知识比框架更重要
4. **为了模式而模式**: 设计模式要在合适的场景使用
5. **忽视性能**: 性能优化应该贯穿整个开发过程

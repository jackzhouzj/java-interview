### Java基础

#### 《Java核心技术》（Core Java）
- **作者**: Cay S. Horstmann
- **难度**: ⭐⭐⭐
- **推荐指数**: ⭐⭐⭐⭐⭐
- **适合人群**: Java初学者、进阶者
- **核心内容**: Java语法、面向对象、集合框架、并发、IO
- **推荐理由**: Java入门经典，内容全面、讲解清晰

## 🎯 实战项目推荐

### 初级项目（适合入门）

#### 1. 博客系统
- **技术栈**: Spring Boot + MyBatis + MySQL
- **核心功能**: 文章发布、评论、标签
- **学习重点**: CRUD操作、权限控制

#### 2. 待办事项系统
- **技术栈**: Spring Boot + Redis + MySQL
- **核心功能**: 任务管理、提醒、分类
- **学习重点**: 缓存使用、定时任务

---

### 中级项目（适合进阶）

#### 1. 电商系统
- **技术栈**: Spring Boot + MyBatis-Plus + Redis + RabbitMQ
- **核心功能**: 商品管理、订单、支付、库存
- **学习重点**: 分布式事务、消息队列、缓存

#### 2. 社交平台
- **技术栈**: Spring Boot + MongoDB + Redis + Kafka
- **核心功能**: 用户关系、动态发布、消息推送
- **学习重点**: NoSQL、实时通信、推荐算法

---

### 高级项目（适合精通）

#### 1. 微服务电商平台
- **技术栈**: Spring Cloud Alibaba + Nacos + Sentinel + Seata
- **核心功能**: 完整的微服务架构
- **学习重点**: 微服务治理、分布式事务、链路追踪

#### 2. 大数据分析平台
- **技术栈**: Spring Boot + Kafka + Flink + Elasticsearch
- **核心功能**: 数据采集、实时计算、可视化
- **学习重点**: 流式计算、大数据处理

# Java核心

## 📋 模块概述

本模块涵盖Java语言的核心知识，包括Java基础语法、面向对象编程、JVM原理等内容。这是Java后端开发的基石，必须扎实掌握。

## 📚 学习内容

### 1. Java基础
- Java语言基础语法
- JDK 8+ 版本特性对比
- 泛型、反射、注解等高级特性

### 2. JVM原理
- JVM内存模型
- 垃圾回收机制
- 类加载机制
- 性能调优

## 🎯 学习目标

- [ ] 掌握Java核心语法和面向对象编程
- [ ] 理解JDK 8+的新特性（Lambda、Stream、Optional等）
- [ ] 深入理解JVM内存模型和垃圾回收机制
- [ ] 掌握JVM性能调优方法

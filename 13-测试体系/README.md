# 13-测试体系

## 📚 模块概述

本模块涵盖Java后端开发中的测试体系，包括单元测试、集成测试、性能测试等内容。完善的测试体系是保证代码质量的关键。

## 🎯 学习目标

- 掌握单元测试的编写方法
- 理解测试驱动开发(TDD)
- 掌握Mock测试技术
- 能够进行集成测试和性能测试
- 建立完整的测试体系

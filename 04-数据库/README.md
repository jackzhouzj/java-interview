# 04-数据库模块

## 📚 模块概述

本模块涵盖Java后端开发中常用的数据库技术，包括关系型数据库（MySQL、Oracle、SQL Server、PostgreSQL）和NoSQL数据库（Redis、MongoDB、Elasticsearch）。

## 🎯 学习目标

- 掌握主流关系型数据库的使用和优化
- 理解NoSQL数据库的特点和应用场景
- 掌握缓存技术和搜索引擎的使用
- 能够根据业务场景选择合适的数据库

## 📊 技术选型建议

### 关系型数据库选择

| 场景 | 推荐数据库 | 理由 |
|------|-----------|------|
| 互联网应用 | MySQL | 开源、性能好、生态完善 |
| 企业级应用 | Oracle | 功能强大、稳定性高 |
| .NET应用 | SQL Server | 与Windows生态集成好 |
| 复杂查询 | PostgreSQL | 功能丰富、符合SQL标准 |

### NoSQL数据库选择

| 场景 | 推荐数据库 | 理由 |
|------|-----------|------|
| 缓存 | Redis | 性能极高、数据类型丰富 |
| 文档存储 | MongoDB | Schema灵活、易于扩展 |
| 全文搜索 | Elasticsearch | 搜索性能强、分析功能丰富 |

## 💡 学习建议

1. **MySQL和Redis是重点**：这两个数据库在实际项目中使用最频繁，必须深入掌握
2. **理论结合实践**：每学完一个知识点，都要动手实践
3. **关注性能优化**：数据库性能优化是面试和实际工作的重点
4. **了解原理**：不仅要会用，还要理解底层原理
5. **对比学习**：对比不同数据库的特点和适用场景

# 03-持久层框架

## 📚 模块概述

持久层框架是Java后端开发中连接应用程序和数据库的关键组件。本模块涵盖主流的持久层框架，帮助开发者掌握数据访问层的设计与实现。

## 🎯 学习目标

- 掌握MyBatis的核心特性和使用方法
- 理解MyBatis-Plus的增强功能
- 了解Hibernate/JPA规范和实现
- 能够根据项目需求选择合适的持久层框架

## 📊 框架对比

| 特性 | MyBatis | MyBatis-Plus | Hibernate/JPA |
|------|---------|--------------|---------------|
| SQL控制 | 完全控制 | 完全控制 | 部分控制 |
| 学习曲线 | 中等 | 简单 | 较陡 |
| 开发效率 | 中等 | 高 | 高 |
| 灵活性 | 高 | 高 | 中等 |
| 性能优化 | 容易 | 容易 | 较难 |
| 适用场景 | 复杂SQL | 快速开发 | 标准化项目 |

## 🎓 前置知识

- Java基础语法
- JDBC基础
- SQL语句编写
- Spring Framework基础

## 📝 学习建议

1. **先学MyBatis**: 作为最基础的持久层框架，理解其核心原理
2. **再学MyBatis-Plus**: 在MyBatis基础上学习增强功能
3. **对比学习JPA**: 了解不同持久层框架的设计思想
4. **实战练习**: 通过项目实践巩固所学知识

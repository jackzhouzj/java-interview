# 17-实战项目

## 📚 模块概述

本模块提供完整的实战项目案例，涵盖电商、秒杀、IM、支付等典型业务场景，帮助你将所学技术应用到实际项目中。

## 🎯 学习目标

- 掌握完整项目的开发流程
- 理解典型业务场景的技术方案
- 能够独立设计和实现复杂系统
- 积累项目经验和解决问题的能力

## 📖 项目列表

### 1. 电商系统
- **难度**: ⭐⭐⭐⭐⭐
- **技术栈**: Spring Boot + MyBatis-Plus + Redis + RabbitMQ + MySQL
- **核心功能**:
  - 商品管理
  - 订单系统
  - 购物车
  - 支付系统
  - 库存管理
  - 优惠券系统

### 2. 秒杀系统
- **难度**: ⭐⭐⭐⭐⭐
- **技术栈**: Spring Boot + Redis + Kafka + MySQL
- **核心功能**:
  - 高并发秒杀
  - 库存扣减
  - 限流防刷
  - 订单异步处理
  - 缓存预热

### 3. IM即时通讯
- **难度**: ⭐⭐⭐⭐⭐
- **技术栈**: Spring Boot + Netty + WebSocket + MongoDB
- **核心功能**:
  - 单聊群聊
  - 消息推送
  - 离线消息
  - 文件传输
  - 消息已读未读

### 4. 支付系统
- **难度**: ⭐⭐⭐⭐⭐
- **技术栈**: Spring Boot + MyBatis + RocketMQ + MySQL
- **核心功能**:
  - 支付接口对接
  - 订单管理
  - 对账系统
  - 退款处理
  - 分布式事务

### 5. 短链系统
- **难度**: ⭐⭐⭐⭐
- **技术栈**: Spring Boot + Redis + MySQL
- **核心功能**:
  - 短链生成
  - 短链跳转
  - 访问统计
  - 防刷机制
  - 高并发优化

### 6. 博客系统
- **难度**: ⭐⭐⭐
- **技术栈**: Spring Boot + MyBatis-Plus + Redis + Elasticsearch
- **核心功能**:
  - 文章管理
  - 评论系统
  - 标签分类
  - 全文搜索
  - 用户系统

### 7. 权限管理系统
- **难度**: ⭐⭐⭐⭐
- **技术栈**: Spring Boot + Spring Security + MyBatis-Plus + Redis
- **核心功能**:
  - RBAC权限模型
  - 用户管理
  - 角色管理
  - 菜单管理
  - 数据权限

### 8. 任务调度平台
- **难度**: ⭐⭐⭐⭐
- **技术栈**: Spring Boot + XXL-Job + MySQL
- **核心功能**:
  - 任务管理
  - 任务调度
  - 任务监控
  - 失败重试
  - 任务分片

### 9. 监控告警系统 ⭐ 重难点
- **难度**: ⭐⭐⭐⭐⭐
- **技术栈**: Spring Boot + Prometheus + Grafana + SkyWalking + AlertManager
- **核心难点**:
  - 海量指标采集与存储（百万级/秒）
  - 分布式链路追踪（跨服务调用链路）
  - 智能告警降噪（告警聚合、自动自愈）

### 10. API网关系统 ⭐ 重难点
- **难度**: ⭐⭐⭐⭐⭐
- **技术栈**: Spring Cloud Gateway + Redis + Nacos + Sentinel
- **核心难点**:
  - 高性能路由（10万+QPS、灰度发布）
  - 分布式限流（多维度限流、集群限流）
  - 统一认证鉴权（JWT、细粒度权限）

### 11. 配置中心系统 ⭐ 重难点
- **难度**: ⭐⭐⭐⭐⭐
- **技术栈**: Spring Cloud Config + Nacos + Apollo + Git
- **核心难点**:
  - 配置热更新（实时推送、无感知刷新）
  - 配置灰度发布（分批发布、实时监控）
  - 配置一致性保证（Raft协议、版本管理）

### 12. 推荐系统 ⭐ 重难点
- **难度**: ⭐⭐⭐⭐⭐
- **技术栈**: Spring Boot + Redis + Flink + Elasticsearch + TensorFlow
- **核心难点**:
  - 实时特征计算（毫秒级特征计算）
  - 多路召回融合（协同过滤、内容推荐）
  - 模型在线服务（深度学习模型推理）

## 🗺️ 学习路线

### 阶段一：基础项目（1-2个月）
1. 博客系统 - 掌握基础CRUD
2. 权限管理系统 - 理解权限设计

### 阶段二：进阶项目（2-3个月）
1. 电商系统 - 理解复杂业务
2. 短链系统 - 掌握高并发优化

### 阶段三：高级项目（3-4个月）
1. 秒杀系统 - 极致性能优化
2. IM系统 - 长连接和实时通信
3. 支付系统 - 分布式事务
4. 数据同步系统 - 实时数据同步
5. 搜索系统 - 亿级数据搜索

### 阶段四：架构级项目（4-6个月）
1. 监控告警系统 - 全方位监控
2. API网关系统 - 统一流量入口
3. 配置中心系统 - 配置统一管理
4. 推荐系统 - 个性化推荐

## 💡 学习建议

1. **循序渐进**: 从简单项目开始，逐步提升难度
2. **完整实现**: 不要只做核心功能，要完成完整的项目
3. **代码规范**: 遵循代码规范，注重代码质量
4. **文档完善**: 编写完整的项目文档和README
5. **持续优化**: 项目完成后持续优化和重构
6. **总结沉淀**: 每个项目完成后总结技术方案和经验

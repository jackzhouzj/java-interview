# 15-接口设计

## 📚 模块概述

本模块涵盖Java后端接口设计的最佳实践，包括RESTful API设计、接口文档、API网关等内容。

## 🎯 学习目标

- 掌握RESTful API设计规范
- 理解GraphQL
- 掌握接口文档工具
- 能够设计高质量的API
- 掌握API版本管理

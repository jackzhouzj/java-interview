# 10-大数据与AI

## 📋 模块概述

本模块涵盖Java后端开发中常用的大数据技术和AI集成方案，帮助开发者拓展技术视野，掌握大数据处理和AI应用的核心技能。

## 📚 学习内容

### 大数据技术
- **Hadoop**: 分布式存储与计算框架
- **Spark**: 内存计算引擎
- **Flink**: 流处理框架
- **Hive**: 数据仓库工具

### AI集成
- **Spring AI**: Spring生态的AI集成方案
- **LangChain4j**: Java版LangChain，LLM应用开发框架 🔥
- **向量数据库**: Pinecone、Milvus等向量存储方案 🔥

## 🎯 学习目标

- 理解大数据技术的核心原理和应用场景
- 掌握Hadoop生态的基本使用
- 了解流处理和批处理的区别
- 学习如何在Java后端集成AI能力

## 📖 学习路径

```
Hadoop基础 → Spark计算 → Flink流处理 → Hive数据仓库 → Spring AI集成
```

## ⚠️ 学习建议

1. **前置知识**: 需要掌握Java基础、Linux基本操作、分布式系统概念
2. **学习重点**: 重点理解分布式存储和计算的原理，而非深入每个组件的细节
3. **实践为主**: 大数据技术需要实际搭建环境进行实践
4. **与Java集成**: 关注如何在Java后端项目中集成和使用这些技术

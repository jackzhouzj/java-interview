# 数据库操作

## 📋 模块概述

本模块涵盖Go语言数据库操作的核心技术，包括MySQL、GORM框架、Redis、MongoDB等数据库的操作方法。

## 📚 学习内容

### 1. MySQL操作
- database/sql包
- 连接管理
- CRUD操作
- 事务处理
- 预处理语句

### 2. GORM框架
- GORM基础
- 模型定义
- CRUD操作
- 关联查询
- 事务和钩子

### 3. Redis操作
- go-redis客户端
- 基本数据类型
- 发布订阅
- 管道和事务
- 分布式锁

### 4. MongoDB操作
- mongo-driver
- 文档操作
- 聚合查询
- 索引管理
- 事务支持

### 5. 数据库连接池
- 连接池原理
- 参数配置
- 性能优化
- 监控和调试

## 🎯 学习目标

- [ ] 掌握MySQL基本操作

- [ ] 掌握Redis操作

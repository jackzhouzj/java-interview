# 微服务治理

> @author erik.zhou

## 📚 模块概述

本模块涵盖微服务架构中的核心组件，包括配置中心、服务网关、注册中心、分布式事务、分布式锁和限流降级等。

## 📖 学习内容

### [01-配置中心](./01-配置中心/)
- [Nacos](./01-配置中心/Nacos/) - 服务发现与配置管理
- [Apollo](./01-配置中心/Apollo/) - 分布式配置中心
- [Spring Cloud Config](./01-配置中心/Spring-Cloud-Config/) - Spring原生配置中心

### [02-服务网关](./02-服务网关/)
- [Spring Cloud Gateway](./02-服务网关/Spring-Cloud-Gateway/) - 响应式网关

### [03-注册中心](./03-注册中心/)
- [ZooKeeper](./03-注册中心/ZooKeeper/) - 分布式协调服务

### 04-分布式事务
- [Seata完整教程](./Seata-完整教程.md) - 分布式事务解决方案（AT/TCC/Saga/XA模式）

### [05-分布式锁](./05-分布式锁/)
- [Redisson](./05-分布式锁/Redisson/) - Redis分布式锁

### [06-限流降级](./06-限流降级/)
- [Sentinel](./06-限流降级/Sentinel/) - 流量控制与熔断降级

## 🎯 学习目标

- [ ] 掌握微服务配置管理
- [ ] 理解服务网关的作用和原理
- [ ] 掌握分布式事务解决方案
- [ ] 能够实现服务限流和熔断降级

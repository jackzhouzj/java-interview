# Spring生态

> @author erik.zhou

## 📚 模块概述

本模块涵盖Spring全家桶的核心技术，包括Spring Framework、Spring Boot、Spring MVC、Spring Cloud等，是Java企业级开发的核心框架。

## 📖 学习内容

### [01-Spring-Framework](./01-Spring-Framework/)
- IoC容器与依赖注入
- AOP面向切面编程
- 事务管理
- Bean生命周期

### [02-Spring-Boot](./02-Spring-Boot/) 🔥
- 自动配置原理
- Starter机制
- 配置管理
- Actuator监控

### [03-Spring-MVC](./03-Spring-MVC/)
- 请求处理流程
- 参数绑定与数据校验
- 拦截器与过滤器
- RESTful API设计

### [04-Spring-WebFlux](./04-Spring-WebFlux/)
- 响应式编程
- Reactor框架
- 背压机制

### [05-Spring-Cache](./05-Spring-Cache/)
- 缓存注解
- 缓存管理器
- 缓存策略

### [06-Spring-Cloud](./06-Spring-Cloud/)
- 服务注册与发现
- 配置中心
- 服务网关
- 负载均衡

### [07-Spring-Cloud-Alibaba](./07-Spring-Cloud-Alibaba/)
- Nacos集成
- Sentinel集成
- Seata集成

## 🎯 学习目标

- [ ] 掌握Spring IoC和AOP原理
- [ ] 能够独立搭建Spring Boot项目
- [ ] 理解Spring MVC请求处理流程
- [ ] 掌握Spring Cloud微服务开发

- **Spring Boot**: 30-40小时
- **Spring MVC**: 20-30小时
- **Spring Cloud**: 35-45小时
- **总计**: 110-150小时

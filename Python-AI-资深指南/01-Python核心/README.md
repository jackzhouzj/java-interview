# Python核心

## 📋 模块概述

本模块涵盖Python语言的核心知识，包括Python基础语法、进阶特性、异步编程等内容。这是Python-AI开发的基石，必须扎实掌握。

## 📚 学习内容

### 1. Python基础
- Python语言基础语法
- 数据类型和数据结构
- 函数和模块
- 面向对象编程
- 文件操作和异常处理

### 2. Python进阶
- 装饰器和生成器
- 上下文管理器
- 迭代器和可迭代对象
- 元类和反射
- 函数式编程

### 3. 异步编程
- asyncio基础
- async/await语法
- 协程和任务
- 异步IO操作

## 🎯 学习目标

- [ ] 掌握Python核心语法和编程思想
- [ ] 理解面向对象编程和函数式编程
- [ ] 掌握Python高级特性（装饰器、生成器等）
- [ ] 能够使用异步编程处理并发任务

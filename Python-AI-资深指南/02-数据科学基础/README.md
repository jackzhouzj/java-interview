# 数据科学基础

## 📋 模块概述

本模块涵盖数据科学的核心工具和技术，包括NumPy数值计算、Pandas数据分析、数据可视化、数据预处理等内容。这是进入AI领域的必备基础。

## 📚 学习内容

### 1. NumPy
- 数组创建和操作
- 数组运算和广播机制
- 索引和切片
- 线性代数运算
- 随机数生成

### 2. Pandas
- DataFrame和Series
- 数据读取和写入
- 数据清洗和转换
- 分组聚合和透视表
- 时间序列处理

### 3. 数据可视化
- Matplotlib基础绘图
- Seaborn统计可视化
- 图表类型和美化
- 交互式可视化

### 4. 数据预处理
- 缺失值处理
- 异常值检测和处理
- 数据标准化和归一化
- 特征编码（独热编码、标签编码）

## 🎯 学习目标

- [ ] 掌握NumPy进行高效数值计算
- [ ] 熟练使用Pandas进行数据分析
- [ ] 能够创建各种数据可视化图表
- [ ] 理解并实践数据预处理流程

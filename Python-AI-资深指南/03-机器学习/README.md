# 机器学习

## 📋 模块概述

本模块涵盖机器学习的核心算法和应用，包括Scikit-learn框架、经典机器学习算法、特征工程、模型评估与调优等内容。这是AI开发的重要基础。

## 📚 学习内容

### 1. Scikit-learn
- 监督学习（分类、回归）
- 无监督学习（聚类、降维）
- 模型评估和选择
- 管道和网格搜索
- 模型持久化

### 2. 机器学习算法
- 线性回归、逻辑回归
- 决策树、随机森林
- 支持向量机（SVM）
- K近邻（KNN）
- 朴素贝叶斯
- 集成学习（Bagging、Boosting）

### 3. 特征工程
- 特征选择方法
- 特征提取技术
- 特征构造策略
- 降维技术（PCA、t-SNE）
- 特征重要性分析

### 4. 模型评估与调优
- 交叉验证
- 评估指标（准确率、精确率、召回率、F1等）
- 超参数调优（网格搜索、随机搜索）
- 模型融合技术
- 过拟合和欠拟合处理

### 5. 高级算法
- XGBoost
- LightGBM
- CatBoost

## 🎯 学习目标

- [ ] 理解常见机器学习算法的原理和应用场景
- [ ] 熟练使用Scikit-learn构建和评估模型
- [ ] 掌握特征工程的核心技巧
- [ ] 能够进行模型调优和性能优化
- [ ] 了解集成学习和高级算法

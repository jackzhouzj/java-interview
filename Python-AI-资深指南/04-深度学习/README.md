# 深度学习

## 📋 模块概述

本模块涵盖深度学习的核心框架和技术，包括PyTorch、TensorFlow、神经网络基础、CNN、RNN、Transformer等内容。这是现代AI应用的核心技术。

## 📚 学习内容

### 1. PyTorch
- 张量操作和自动微分
- 神经网络模块（nn.Module）
- 优化器和损失函数
- 数据加载和预处理
- 模型训练和验证
- 模型保存和加载

### 2. TensorFlow/Keras
- TensorFlow基础
- Keras高级API
- 模型构建方式
- 自定义层和模型
- 回调函数和监控

### 3. 神经网络基础
- 感知机和多层感知机
- 反向传播算法
- 激活函数（ReLU、Sigmoid、Tanh等）
- 正则化技术（Dropout、L1/L2）
- 批归一化（Batch Normalization）
- 优化算法（SGD、Adam、RMSprop等）

### 4. CNN卷积神经网络
- 卷积层和池化层
- 经典网络架构（LeNet、AlexNet、VGG、ResNet）
- 迁移学习
- 图像分类应用
- 数据增强技术

### 5. RNN循环神经网络
- RNN基础原理
- LSTM和GRU
- 序列建模
- 时间序列预测
- 文本生成

### 6. Transformer
- 注意力机制（Attention）
- Self-Attention和Multi-Head Attention
- Transformer架构
- 位置编码
- BERT和GPT原理

## 🎯 学习目标

- [ ] 熟练使用PyTorch构建和训练神经网络
- [ ] 理解CNN、RNN、Transformer的原理和应用
- [ ] 掌握迁移学习和模型微调技术
- [ ] 能够解决实际的深度学习问题
- [ ] 了解深度学习的最佳实践
